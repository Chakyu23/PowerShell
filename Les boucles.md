# Les Boucle

## Qu'est ce qu'une boucle 

Une boucle est une bonne manière de répété une action en posant des condition spécifique.

## Comment réalisé un boucle 

La boucle de base est une boucle Jusqu'a 
* While
* do
* *mettez vos action ici*
* done
